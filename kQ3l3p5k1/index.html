使用Gridea+Github Pages搭建个人博客并绑定域名

2021-01-03 / 2 min read

搭建个人静态博客

创建GitHub Pages仓库

在GitHub首页，点击+号->New repository

仓库的名称必须是用户名.github.io（我已经创建过这个仓库，所以会报错）

设置Gridea

Gridea是一个静态博客的写作客户端

点击此处的链接，如图，下载Gridea

安装完成后，配置Gridex，点击远程，如图所示

令牌可由如下操作生成

Settings -> Developer Settings -> Personal access tokens -> Generate new token

创建成功后，会生成一串token，这串token之后不会再显示，所以要好好保存。

Gridex配置完成后，点击检测远程连接，连接成功后，点击保存。

你可以根据自己的喜好选择主题，具体可参考快速上手

绑定域名

去运营商处购买域名，我是在腾讯云购买，新用户有很大优惠，可以选择一些.xyz、.today之类的便宜域名，我的是25元/年。

新用户需要上传个人信息进行备案，会有提示，跟着操作就行。

在cmd中使用ping命令找领导你的GitHub Page的主机地址，如图所示

进入域名控制台，解析域名

添加两条记录，格式如下图所示，第一条的记录值，为第上面ping得的ip地址

在你的仓库中的Settings中找到Github Pages，如下图设置，点击保存

并在Gridea远程设置中，添加CNAME，点击保存

至此，你的个人博客已经搭建完毕并与域名绑定，你可以通过Gridea方便地进行写作和管理博客。
